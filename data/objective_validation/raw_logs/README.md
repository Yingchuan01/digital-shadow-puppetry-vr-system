# Raw objective validation logs

This folder contains the raw objective validation logs used to generate the processed results.

The raw logs include frame-level trial logs, contact-event logs, trial summaries, parameter settings, threshold settings, and the trial manifest.

## Folder structure

- `04_trial_logs/`: frame-level trial logs
- `05_contact_event_logs/`: contact-event logs
- `06_trial_summary/`: trial-level summary files
- `parameter_settings/`: runtime and analysis parameter settings
- `threshold_settings/`: threshold settings used for metric computation
- `trial_manifest/`: trial manifest files

All raw logs were generated during controlled objective validation sessions under the same hardware, software, and runtime parameter settings.
