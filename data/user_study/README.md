# User study data

This folder contains anonymized user-study materials and results for the subjective validation of the digital shadow-puppetry interaction system.

The user study compared system version 1.0 and system version 2.0 using a within-subject design. Participants experienced both versions and rated the system across five dimensions:

1. Naturalness
2. Controllability
3. Collision realism
4. Visual credibility
5. Overall experience

## Contents

- `digital_shadow_puppetry_user_study_data_anonymized.zip`: anonymized user-study questionnaire materials, exported questionnaire data, processed scores, and statistical results.

## Data privacy

All participant-identifiable information has been removed. The released dataset uses anonymized participant IDs such as `P01`, `P02`, and so on. Names, phone numbers, emails, student IDs, IP addresses, detailed submission timestamps, and other directly identifying information are not included.
