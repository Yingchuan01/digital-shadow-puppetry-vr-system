# Digital Shadow-Puppetry VR System

This repository provides supplementary data, analysis code, and demonstration materials for a digital shadow-puppetry interaction system.

The system was developed to support real-time body-driven digital shadow-puppetry interaction. It uses a three-layer architecture that separates markerless motion input, planar IK-based retargeting, and physics-mediated contact-preserving rendering.
